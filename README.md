# SprintKey ⌨️

SprintKey is a beginner-friendly typing speed tester built in C++ that runs entirely in the terminal.

## Features

- Difficulty levels:
  - Easy
  - Medium
  - Hard
- Random paragraph selection (right now only 5 paras in each levels)
- Typing timer
- Words Per Minute (WPM) calculation
- Simple terminal interface
- Help section explaining how the program works

## Tech Stack 
C++

## How to Run

- Clone the repository
- Go to the project folder
- Compile
- Run

## How It Works

1. Launch the program.
2. Choose **Start Test**.
3. Select a difficulty level.
4. A random paragraph is displayed.
5. Type the paragraph and press **Enter** when finished.
6. The program calculates:
   - Time Taken
   - Words Per Minute (WPM)

I built SprintKey as my first C++ project after learning the basics of the language. The goal was to combine functions, arrays, strings, randomization, timers, and user input into a complete console application instead of writing small practice programs.
